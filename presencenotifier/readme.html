Presence Notifier Plugin Readme

Overview

This plugin allows notify to an external server when an user goes to online or offline. It could be useful if you need to do searches (or whatever) in your own database out from openfire (a webpage, search engine, etc)

Configuration

You have to configure:
  • Destination url: The url where your external server are listening
  • Token: A secret token that only are known by the server and openfire
The plugin will add as get parameters the token, username and state. The state are represented as "online=true|false"

Example

Destination url: http://miserver.com/chatNotifier
Token: aabjdjk3
The user "enjuto" starts session.
